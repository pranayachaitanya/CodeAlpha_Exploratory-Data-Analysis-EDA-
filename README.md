# CodeAlpha_Exploratory-Data-Analysis-EDA-
🚀 Amazon Product Data Analysis — CodeAlpha Internship

Thrilled to share my internship project at CodeAlpha, where I explored Amazon product data using Python to uncover insights on pricing, discounts, and customer ratings.

🔑 Key Concepts Applied

Data Cleaning & Preprocessing

Exploratory Data Analysis (EDA)

Correlation & Outlier Detection

Hypothesis Testing (Independent Samples T-Test)

Data Visualization with Matplotlib & Seaborn

📊 Insights Gained

Strong correlation (0.96) between actual & discounted prices

Higher discounts often applied to lower-priced items

Outliers detected in premium product segments

Discounts significantly influence customer ratings (p-value < 0.05)

🧠 Conclusion  
This project strengthened my skills in data cleaning, visualization, and statistical inference, showing how structured analysis can drive business insights in e-commerce.
